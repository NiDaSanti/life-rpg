# Life Stats Dashboard

This is a terminal-style RPG stat tracker dashboard. All data is stored in your browser's localStorage—no backend or database required.

## How to Deploy on Netlify

1. Go to https://app.netlify.com/ and sign up or log in.
2. Click "Add new site" > "Import an existing project".
3. Connect your GitHub repo or drag-and-drop this folder.
4. Netlify will auto-detect this as a static site. No build command is needed.
5. Click "Deploy site". Your dashboard will be live!

## Files
- `life.html` — Main dashboard UI
- `index.html` — Netlify entry point (redirects to life.html)
- `index.js` — Stat tracking logic
- `styles.css` — Terminal-inspired styling
- `favicon.svg` — Stats favicon
- `netlify.toml` — Netlify config
- `package.json` — Project metadata

## Features
- Animated stat displays
- Habit and stat tracking
- Log history and achievements
- Terminal-style modal and UI
- LocalStorage only (no backend)
